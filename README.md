# 砖头插件 (AstrBot)

由 Koishi 砖头插件迁移而来，支持更多的趣味互动功能。

## 功能列表

1.  **烧砖 (`/烧砖`)**: 开始烧制砖块。群友每发送一定数量的消息，你就能获得一块砖。
2.  **拍人 (`/拍人 @用户`)**: 使用一块砖头尝试拍晕（模拟禁言）对方随机时间。小心！对方有概率夺过砖头反拍你。
3.  **随机拍人 (`/随机拍人`)**: 随机挑选一名幸运群友使用砖头。
4.  **查看砖头 (`/查看砖头`)**: 看看你手里现在攒了多少砖头。
5.  **砖头签到 (`/砖头签到`)**: 每日领取免费砖头。
6.  **偷砖 (`/偷砖 @用户`)**: 尝试偷取别人的砖头。有概率失败，失败会被“扭送公安局”（禁言）。
7.  **搬砖 (`/搬砖`)**: 开启搬砖任务，通过群聊活跃度快速获取大量砖头，但有概率“累晕”（禁言）。
8.  **黑名单功能**:
    *   **个人黑名单 (`/别拍我了`)**: 开启后，插件将不再响应你的任何指令，且别人也无法对你进行操作。开启需确认，一旦开启无法自主取消。
    *   **群黑名单 (`/禁砖头` / `/开启砖头`)**: 群管理或机器人管理员可禁用/开启本群的砖头功能。

## 配置项

管理员可以在 Web 控制面板或通过指令 `/砖头配置` 修改以下参数：

*   `max_brick`: 砖块持有上限。
*   `cost`: 烧砖所需消息数。
*   `cooldown`: 拍人冷却时间。
*   `reverse_prob`: 反拍概率。
*   `enable_steal`: 是否开启偷砖。
*   `steal_fail_prob`: 偷砖失败概率。
*   `steal_range`: 偷砖数量区间。
*   `enable_work`: 是否开启搬砖。
*   `work_fail_prob`: 搬砖累晕概率。
*   `enable_checkin`: 是否开启签到。
*   `enable_user_blacklist`: 是否开启用户黑名单。
*   `enable_guild_blacklist`: 是否开启群黑名单。

## 安装

将插件文件夹 `astrbot_plugin_brick` 放入 AstrBot 的 `plugins` 目录下即可。
